# Budget Stack

Budget Stack is a simple, privacy-focused budgeting and purchase planning app for iPhone.

## Features

- Create and organize purchase lists
- Track spending against budgets
- Categorize items with custom tags
- Real-time totals
- iCloud sync
- No analytics or tracking

## Privacy

Budget Stack does not collect personal data and does not track user activity. Nothing is stored by OpenAI, Codex, or Andrew Williams (developer).
